## User Story

**Como** gestante,
**quero** escolher quais tipos de lembrete desejo receber e em que horário,
**para que** o app notifique apenas o que é relevante para mim.

---

## Acceptance Criteria

- [ ] **AC1** — Nova seção "Notificações" acessível via aba Configurações
- [ ] **AC2** — Toggles por tipo: consultas pré-natais, marcos semanais, kick counter, contrações
- [ ] **AC3** — Seletor de horário padrão para lembretes diários (ex: "08:00")
- [ ] **AC4** — Configurações persistidas em SQLite via nova tabela `notification_settings` (criada via `runMigrations()`)
- [ ] **AC5** — Schema: `notification_settings(id INTEGER, type TEXT, enabled INTEGER, default_time TEXT)`
- [ ] **AC6** — Estado inicial: todos os lembretes desativados (usuário ativa explicitamente)
- [ ] **AC7** — Na plataforma web: seção exibida mas desabilitada com mensagem "Notificações disponíveis apenas no app mobile"
- [ ] **AC8** — `npm run typecheck` zero erros
