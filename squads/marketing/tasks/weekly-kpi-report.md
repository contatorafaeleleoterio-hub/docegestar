# Weekly KPI Report

Tarefa para consolidar e analisar as metricas semanais da squad de marketing.

## Procedimento
1. Coletar dados de Google Ads (conversoes, CPC, gasto).
2. Coletar dados da Play Store (impressoes, downloads organicos, taxa de conversao).
3. Comparar os resultados com a semana anterior e metas estabelecidas.
4. Identificar tendencias e oportunidades de otimizacao.
5. Gerar o relatorio em markdown para visualizacao da Gabriela.

## Checklist
- [ ] Dados de Ads coletados.
- [ ] Dados da Play Store coletados.
- [ ] Analise comparativa realizada.
- [ ] Relatorio markdown gerado.
